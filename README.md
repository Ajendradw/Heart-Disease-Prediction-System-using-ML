# ❤️ HeartCare AI

## 📌 Overview

HeartCare AI is a Machine Learning web application that predicts the risk of heart disease based on medical parameters such as age, blood pressure, cholesterol, ECG results, and more.

The application is built using Python, Scikit-learn, and Streamlit.

---

## 🚀 Features

- Heart disease prediction
- Interactive web interface
- Real-time prediction
- Machine Learning powered
- KNN classification model
- Scaled and processed input data

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/heartcare-ai.git
```

### Move into project folder

```bash
cd heartcare-ai
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
